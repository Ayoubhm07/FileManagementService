# Minio File Manager Service

This project is a Spring Boot application that integrates with Minio, a high-performance object storage system, to manage file uploads and downloads.

## Features

- Upload files to Minio
- Download files from Minio
- List files in a Minio bucket
- Delete files from Minio

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- Java 11 or higher
- Maven
- Minio server
- Git
